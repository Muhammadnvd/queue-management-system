This is a simple Queue Processing System made in C++ using Linked List.

Customers are added into two types of queues based on the number of grocery items:

Express Queue → 5 items or less

Normal Queue → More than 5 items

✅ Features

Add customer (ID, Name, Items)

Automatically assigns customer to Express or Normal queue

Displays customers queue-wise

Removes customers after processing

🛠️ Technologies

C++

Linked List (Data Structures)

▶️ How to Run

Compile the program

Run and enter customer details

System will display Normal queue first, then Express queue
